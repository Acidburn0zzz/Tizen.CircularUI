    Radio

    Radio displays 1 or more option and allows a user to select 1 of them. Radio can select only 1 item among items having same GroupName property.

    Create Radio

    GroupNameproperty specifies which Radio controls are mutually exclusive. If a user selected one Radio control of radio control group, other items are not selected. IsSelected property sets default selected value of Radio control. Selected event occurs when the Radio selection was changed. Value property sets any value of Radio control. Value property is useful to distinguish which item was selected if all Radio control has the same Selected event handler. Please refer to below code.

    For more information. Please refer to Radio API reference

    This guide's code example uses WearableUIGallery's TCRadio code at the test\WearableUIGallery\WearableUIGallery\TC\TCRadio.xaml.cs

            public void OnSelected(object sender, SelectedEventArgs args)
        {
            Console.WriteLine($"OnSoundSelected!! value:{args.Value}");
            Radio radio = sender as Radio;
            if (radio != null)
            {
                if (args.Value) label.Text = "Selected : " + radio.Value;
            }
        }
